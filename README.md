  # Guitar Fretboard Scale Visualizer

  A lightweight web tool for guitar scale visualization, route practice, numbered notation, octave dots, and manual note
  highlighting.

  ## Features

  - Horizontal six-string guitar fretboard
  - Major, major pentatonic, and major blues scales
  - Numbered notation with octave dots
  - Range and route modes
  - Manual note highlighting
  - English and Traditional Chinese UI
  - Click-to-play note preview

  ## Usage

  Open `index.html` in a browser.
